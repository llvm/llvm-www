Social events

A number of local communities are joining on a more or less regular basis:

  • Austin, USA
  • Bangalore, India
  • Berlin, Germany
  • Brussels, Belgium
  • Chicago, USA
  • London / Cambridge, UK
  • Paris, France
  • Saint Pertersburg, Russia
  • San Francisco bay area, USA
  • Sydney, Australia
  • Stockholm, Sweden
  • Toronto, Canada
  • Zurich, Switzerland

The meetups are usually announced on the various LLVM and Clang mailing lists.

We are using the meetup.com infrastructure as it allows to have all meetups visible in the same place, which can be handy when people are moving around the world. If there is a local community in your place and would like to stand up to organize meetups, please reach arnaud <dot> degrandmaison <at> llvm <dot> org so that we can add your group to the network.
